CSV Format Information

The exported CSV file will contain the following columns:

  • order_id - Internal order ID
  • easyship_shipment_id - Easyship shipment ID
  • tracking_number - FedEx tracking number
  • customer_name - Customer full name
  • customer_email - Customer email
  • customer_phone - Customer phone
  • address_line_1 - Primary address
  • address_line_2 - Secondary address
  • city - Destination city
  • state - Destination state/region
  • postal_code - Postal/ZIP code
  • country - Country code
  • carrier - Shipping carrier (FEDEX)
  • service_type - FedEx service type
  • order_status - Order status
  • created_date - Order creation date
